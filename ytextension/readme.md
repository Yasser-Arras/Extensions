# YouTube Toolbox Extension

A small browser extension that adds a draggable toolbox on YouTube for quick access to useful tools and controls.

---

## Description

- Works on Shorts and watch pages  
- Draggable floating toolbox  
- Lightweight and simple  
- Built to handle YouTube’s dynamic pages  

---

## Current Features

- Frame-by-frame stepping (forward / backward)  
- Playback speed control (cycle + scroll)
- Volume Boost
- Screenshot current frame  
- Open Shorts as normal video page  
- Download video  
  - mp4 (left click)  
  - mp3 (right click)  

---

## Installation

1. Download or clone the repo  
2. Go to `chrome://extensions/`  
3. Enable developer mode  
4. Click "Load unpacked"  
5. Select the project folder  

---

## Notes

- Shorts uses multiple video elements, the extension selects the active one  
- Designed to stay simple and responsive
